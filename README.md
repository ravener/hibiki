# Hibiki

[osu!](https://osu.ppy.sh) bot for [Fluxer](https://fluxer.app)

Invite the bot [here](https://web.fluxer.app/oauth2/authorize?client_id=1475787576686872332&scope=bot&permissions=4503599627750464)

## Running

* Copy `.env.example` to `.env` and fill in the fields.
* You may need to modify some constants in [`src/lib/constants.ts`](src/lib/constants.ts)
* Install dependencies with `pnpm install`
* Build TypeScript with `pnpm run build`
* Start bot with `pnpm run start`
